# Backend I — Entrega N°2  
## Handlebars + WebSockets + Actualización en Tiempo Real

Este proyecto implementa un servidor basado en **Node.js** y **Express**, incorporando:

- Motor de plantillas **Handlebars**
- Comunicación en tiempo real con **Socket.io**
- Endpoints de productos y carritos (Entrega N°1)
- Vistas dinámicas renderizadas desde el servidor
- Actualización automática de la lista de productos sin necesidad de recargar la página

---

## Tecnologías utilizadas

- Node.js  
- Express  
- Express-Handlebars  
- Socket.io  
- FS (File System para persistencia en JSON)
